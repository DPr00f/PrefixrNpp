PrefixrNpp by DPr00f

A Prefixr plugin for Notepad++

Prefixr for Netbeans is a plugin developed by Joao Lopes that uses the website http://prefixr.com/ created by Jefrey Way

Install

1. Download plugin here
2. Close Notepad++ if opened
3. Locate the Notepad++ plugins folder. (Normally %PROGRAMFILES(x86)%\Notepad++\plugins\ or %PROGRAMFILES%\Notepad++\plugins\)
4. Locate Prefixr.dll file and copy it to the plugins folder.
5. Start Notepad++.
